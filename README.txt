Web App displaying analysis of US real estate markets at the county level against variabls such as crime rate and demographics. Tools Used: AWS RDS, MySQL, Node.js, Express.js, Angular.js, HTML/CSS, Python (pandas)

Please follow the below instruction to run this project.
- Extract the zip file 
- Open a new terminal window
- After entering the extracted file, type the following commands
  1 - cd NodeApp
  2 - npm install
  3 - node app
The second command will install all the packages used in this project.
The third will run the server.
To check if the server is actually running, open a browser and go to the following link
http://localhost:8081

You can find the data cleaning scripts in the "data_cleaning" directory.
